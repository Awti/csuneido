Suneido March 20, 2006 Release

Compatibility

  • NOTE: the database format has changed you must dump with the old exe and load with the new one
  • Final() and Memory() are gone (but MemoryArena() is still available)

Maximum database size is now 16gb (instead of roughly 2gb)

Heap size is no longer limited to 64mb (changed to the Boehm-Demers-Weiser garbage collector)

Suneido.exe is now built with the free Microsoft Visual C++ Toolkit 2003 which has reduced exe size and improved speed

Automatic timestamp fields - a field ending with "_TS " will automatically be set to a new timestamp every time a record is saved. Useful for tracking a "last updated" date/time and for a "version number" for checking if another user has updated the record.

Windows Data Execution Protection (DEP) can now be enabled without problems.

Added index checking to suneido -check

Added tables to Wiki.

GUI redraw improvements to reduce flashing (from Jos Schaars)

AccessNavControl - new (from Jos Schaars)
