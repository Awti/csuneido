Suneido April 17, 2005 Release

Plugins

  • The IDE, Refactor, and Help menus are now plugins
  • You can add menu options to any window using plugins
  • You can add these menus to your own windows
  • You can add items to these menus

Refactoring

  • Convert Function to Class - converts a function to a class with a CallClass method

  • Extract Method - automatically determines inputs and output from the code segment

  • Refactorings are now undo-able

Go To's

  • Ctrl+F3 and Shift+Ctrl+F3 find the next or previous occurrence of the selected text (or the current word if you don't have a selection)
  • Go To Definition now works with methods. It will even find the definition of methods in superclasses. If the method isn't found you have the option to automatically create it (handy for "programming by intention").

  • For DLL's Go To Documentation will now use Google to bring up MSDN documentation
  • Go To Random Line of Code (currently plugged into the Refactor menu)

Testing Tools

  • New MockObject and FakeObject classes
  • New Run Failed button on TestRunner

  • Added Create Test for Method (on LibraryView Tools menu)

  • Run Associated Test (on LibraryView tab right-click context menu) now gives option to create the associated test if it doesn't exist

Design Patterns

  • New Singleton abstract base class for implementing the "singleton" design pattern
  • New MockObject class to help write "unit tests"
  • New FakeObject class useful for tests or for implementing the "null object" or "special case" design patterns
  • New ServerEvalProxy for writing classes that exeute on the database server via ServerEval

Miscellaneous

  • New MapButtonControl used by AddressControl - brings up Google, Yahoo, or MapQuest map for location

  • New number.SpanishToWords (contributed by Luis Alfredo)
  • CheckBoxControls can now have tooltips

  • Improved auto-indent in ScintillaControl (e.g. in LibraryView)
  • Added string.White? and string.LineCount
  • New UrlEncode
  • Refresh delays in LibraryView and BookEdit can be controlled by redefining LibViewRefresh and BookEditRefresh
  • Dutch translation updates thanks to Jos Schaars
  • French translation updates from Jean-Luc Chervais
  • updated to Scintilla 1.63

Application Components

  • Added pessimistic locking to AccessControl

  • Added ability to rename and delete reports from Reporter > Open

  • Added ability to print Select's on Reporter reports

Language

  • try-catch no longer catches "block return"
  • record.Copy() now maintains .New?() status
  • Function?() now returns true for blocks
  • @args now checks for too many arguments (warning: this may uncover errors in existing code)
  • Added Sort!, Reverse!, and Partition! (Sort, Reverse, and Partition are deprecated)

Portability

  • GCC version now supports Image and RichEdit (e.g. WordPadControl)
  • GCC now compiles with no warnings
