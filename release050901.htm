Suneido Sept. 1, 2005 Release

Compatibility

  • NOTE: the database format has changed you must dump with the old exe and load with the new one
  • Database.Margin() is gone, and the corresponding +# command line option
  • Date(string) now defaults to yMd rather than the Windows setting (but DateControl still uses Windows setting)
  • ButtonControl now trims its name
  • Browse GetField, SetField, GetCurrentRecord now throw exceptions if there is no current record
  • the argument to Access_RecordChanged is now a list of members rather than a single member name

Database margin is no longer a fixed size. i.e. he amount the database can grow during one session is not limited.

Date.Format now translatable

Suneido.Language = #(name: "french", charset: "DEFAULT")
Date().LongDate()
    => "mardi, août 30, 2005"

AccessBrowseControl - mostly functional but not well tested

BrowseFlipFormControl - only tested when embedded/linked in an AccessControl

Reporter - new Summarize facility

LibraryView > Tools > Options

BookEdit > Tools > Options

IDE > Reset Plugins - new

IDE > Show Plugins - new

CustomizableControl - allows end users to add fields to a table and design a screen layout for them
